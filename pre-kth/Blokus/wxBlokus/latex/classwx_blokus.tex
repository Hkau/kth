\subsection{Detailed Description}
Main frame for Blokus. 

Main window. Contains \doxyref{Rules}{p.}{class_rules} and \doxyref{Game}{p.}{class_game} for gameplay mechanics. A \doxyref{Field\-Panel}{p.}{class_field_panel}, and a few instances of \doxyref{Tile\-Panel}{p.}{class_tile_panel} show the previous, current and next \doxyref{Tile}{p.}{struct_tile}. This frame also handles key events from child windows and have a couple of commands from menus that allows players save, load, skip one's turn etc.
